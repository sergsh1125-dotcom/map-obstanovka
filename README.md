# ☢️ ПЛАТФОРМА ПІДТРИМКИ РХБ ЗАХИСТУ

Інтерактивна веб-платформа для відображення та аналізу радіаційної та хімічної обстановки з можливістю нанесення точок вимірювання на карту.

---

## 🚀 Можливості

- 🗺 Інтерактивна карта (OpenStreetMap + супутник)
- 📍 Нанесення точок вимірювання:
  - кліком по карті
  - вручну
  - із CSV-файлу
- ☢ Радіоактивне забруднення
- ☣ Хімічне забруднення
- 🎯 Відображення значень із підписами
- 🎨 Кольорове кодування:
  - синій — радіоактивне забруднення
  - чорний — хімічне забруднення
- 📊 Таблиця всіх точок
- 💾 Завантаження:
  - карти (HTML)
  - даних (CSV)
- 🧹 Очищення карти
